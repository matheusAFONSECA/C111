# C111 - Análise de Dados

Este repositório é dedicado ao armazenamento e organização dos códigos desenvolvidos na disciplina **C111 - Análise de Dados**, oferecida pelo **Instituto Nacional de Telecomunicações (INATEL)**. O objetivo é centralizar todo o material prático produzido durante a matéria, facilitando o acesso e a revisão dos conteúdos abordados.

## Estrutura do Repositório

O repositório está organizado em duas principais pastas:

### 1. Aulas

Nesta pasta, estão todos os códigos desenvolvidos durante as aulas, assim como os exercícios propostos pelo professor.

### 2. Provas

Aqui estão armazenadas as provas realizadas na disciplina C111. Este conteúdo é útil para revisão e estudo, oferecendo uma visão prática dos conceitos abordados durante a matéria.
